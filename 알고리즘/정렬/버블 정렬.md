# 버블 정렬
데이터의 인접요소끼리 비교하여 swap 연산을 수행하여 정렬하는 기법

### 동작 예시

![[bubble_sort.gif]]

## 코드 구현

```swift
func bubbleSort(_ array: [Int]) {
	for i in 0..<arr.count {
		for j in 0..<arr.count - 1 {
			if arr[j] > arr[j+1] {
				swap(&arr[i],&arr[j+1])
			}
		}
	}
}
```


## 시간 복잡도
$$시간 복잡도 = O(N^2)$$



$$T(n) = (n-1) + (n-2) + (n-3) + \cdots + 2 + 1$$

**1부터 (n−1)까지의 등차수열 합**

---

### 등차수열 합 공식 유도

정방향과 역방향 수열을 나란히 놓으면 각 쌍의 합이 항상 n으로 일정

```
정방향:  1    +  2    +  3    + ⋯ + (n−1)
역방향: (n−1) + (n−2) + (n−3) + ⋯ +  1
─────────────────────────────────────────
열별 합:  n    +  n    +  n    + ⋯ +  n   → (n−1)개
```

$$\text{정방향} + \text{역방향} = n \times (n-1)$$

정방향 하나만 필요하므로 2로 나누면:

$$\sum_{k=1}^{n-1} k = \frac{n(n-1)}{2}$$

---

### 수식 전개

$$T(n) = \frac{n(n-1)}{2} = \frac{n^2 - n}{2} = \frac{1}{2}n^2 - \frac{1}{2}n$$

빅오 표기에서는 **최고차항만 남기고 계수를 제거**하므로:

$$T(n) = O(n^2)$$

---

### 결론

$$\boxed{T(n) = \frac{n(n-1)}{2} \approx O(n^2)}$$

---

### 케이스별 시간복잡도 정리

|케이스|조건|시간복잡도|
|---|---|---|
|최악 (Worst)|역순 정렬 입력|O(n²)|
|평균 (Average)|무작위 입력|O(n²)|
|최선 (Best)|이미 정렬된 입력 + 조기 종료 최적화|O(n)|

> **최선 O(n) 조건:** 한 번의 패스에서 swap이 발생하지 않으면 종료하는 플래그 최적화가 적용된 경우에만 해당된다.  
> 최적화 없는 단순 구현은 최선도 O(n²)이다.
